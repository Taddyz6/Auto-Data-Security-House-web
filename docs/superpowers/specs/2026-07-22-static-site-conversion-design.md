# 纯静态网站改造设计

## 目标

将现有 Next.js 全栈 MVP 改造成可由静态文件托管服务直接部署的网站。保留全部公开内容、首页视觉、动画、响应式布局和六步企业自测，移除后台管理、API、认证、数据库和服务端运行要求。

构建产物必须由 `next build` 输出到 `out/`，可部署到 GitHub Pages、Nginx、对象存储或其他静态托管平台。

## 架构方案

继续使用 Next.js App Router、TypeScript、Tailwind CSS、shadcn/ui 风格组件和 Framer Motion，并在 `next.config.ts` 中启用 `output: "export"`。所有页面在构建期生成 HTML、CSS 和 JavaScript，不包含服务端路由或运行时数据库访问。

公开内容从 Prisma 初始化数据迁移为 TypeScript 常量。页面直接导入这些常量，避免运行时请求。动态场景详情继续通过 `generateStaticParams` 在构建期生成。

## 保留范围

- 首页及动态世界地图、跨境流程动画、能力卡片、场景卡片和对比模块。
- 认识安全屋、全流程演示、应用场景、案例与资质、联系我们、隐私声明和免责声明页面。
- 六步企业自测、现有表单校验、固定评分规则和打印能力。
- 企业信息及孙权、路宽两位联系人。
- FAQ、案例、资质和资料展示能力。
- `prefers-reduced-motion`、移动端响应式和现有视觉系统。

## 移除范围

- `src/app/admin/` 管理员页面。
- `src/app/api/` API 路由。
- `middleware.ts` 管理员访问控制。
- Prisma schema、seed 和 SQLite 数据库。
- 数据库、认证、Cookie、访问统计和线索提交代码。
- Prisma、bcrypt、jose 等仅服务于后端的依赖与脚本。

## 企业自测

自测继续在 `/assessment/` 使用六步表单。第六步校验通过后，浏览器直接调用现有纯函数评分逻辑，不发送网络请求。

结果在当前页面替换或衔接表单展示，包含：

- 需求类型与说明。
- 业务复杂度、敏感关注度、治理缺口和总分。
- 企业类型、方向、场景、数据类型与频率摘要。
- 重点关注事项和推荐下一步。
- 企业联系方式、重新自测和打印按钮。

自测数据仅存在于当前 React 组件内存中。刷新页面后结果清除，不使用 Cookie、`localStorage`、`sessionStorage` 或其他持久化机制。

## 静态内容数据

新增集中式公开内容模块，提供站点设置、案例、FAQ、资质和下载资料的数据类型与固定数据。首页、认识安全屋和案例页面统一读取该模块。

内容模块不得包含虚构法规结论、企业案例、资质或量化效果。模拟能力继续按项目规则标注“演示”或“模拟结果”。

## 资料下载扩展

后续资料文件放入 `public/downloads/`。每个文件在公开内容模块的资料清单中登记以下字段：

- `id`：稳定标识。
- `title`：展示标题。
- `description`：资料说明。
- `fileType`：PDF、DOCX、XLSX 等类型。
- `fileUrl`：以 `/downloads/` 开头的静态路径。

案例与资质页面使用可复用资料卡片渲染清单，并通过普通 `<a download>` 链接下载。没有文件时显示明确空状态，不产生失效链接。新增资料不需要后台或数据库，只需加入文件并更新清单。

## 错误处理

- 自测逐步校验继续使用 Zod 与 React Hook Form。
- 评分异常时保留表单数据并显示可重试提示。
- 下载清单只允许站内 `/downloads/` 路径，避免生成外部或动态下载地址。
- 静态构建必须对缺失动态参数、服务端导入和不支持的路由直接失败。

## 测试与验收

- 保留并扩展评分函数测试，覆盖组件端直接评分的数据结构。
- 增加静态架构回归测试，确保不存在 Prisma、API、管理员路由和网络提交。
- 运行 `npm run lint`、`npm run typecheck`、`npm run test` 和 `npm run build`。
- 验证 `out/` 存在，公开页面均生成静态文件。
- 使用静态 HTTP 服务器打开 `out/`，验证导航、动画、自测同页结果、打印和资料链接。
- 确认刷新自测页面后不保留填写内容或结果。

## 非目标

- 不保留管理员登录、内容编辑、线索查看和 CSV 导出。
- 不保存企业自测、联系方式、访问日志或咨询信息。
- 本轮不加入真实下载资料，仅建立可扩展目录和数据接口。
- 不接入外部 CMS、分析平台或第三方表单服务。
